This folder contains image assets used by the TechTide Solutions website.

- team-sara.png, team-bilal.png, team-ayesha.png, team-hamza.png
  Placeholder avatar images for the "Meet Our Team" section on the About page
  (generated locally as simple initials avatars — no external/copyrighted
  images are used).

No CSS or JavaScript is used anywhere in this project. Card-style layouts
are built with plain HTML tables (border, cellpadding, cellspacing
attributes), and the location map on the Contact page is embedded with a
plain HTML <iframe> — both are valid HTML-only techniques.
